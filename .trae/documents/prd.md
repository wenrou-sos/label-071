## 1. 产品概述

矿井通风系统管理平台是一套面向煤矿安全生产的数字化管理系统，通过实时监控主通风机运行参数、可视化展示矿井通风网络、自动生成通风月报并管理倒机操作与反风演习记录，帮助矿井通风管理人员全面掌握通风系统运行状态，保障矿井安全生产。

- 目标用户：矿井通风工程师、安全管理人员、矿调度室值班人员
- 核心价值：实现通风系统全流程数字化管理，参数超限自动告警与处置，提升通风安全管理效率

## 2. 核心功能

### 2.1 功能模块

1. **通风系统图页面**：以网络图形式展示矿井通风系统，标注巷道风流方向、风量、风压，标记通风构筑物
2. **主通风机监控页面**：实时显示风机运行参数，参数超限告警与停机指令下发，历史告警记录查询
3. **通风月报页面**：自动生成月度通风报表，含用风地点风量对比、有效风量率、风机效率，支持PDF导出
4. **倒机与反风记录页面**：主通风机倒机操作记录、反风演习记录管理

## 3. 核心流程

### 3.1 通风监控告警流程

用户进入主通风机监控页面，系统每5秒刷新实时参数。当轴承温度超过75°C时，系统自动弹出黄色预警通知；超过85°C时，弹出红色报警弹窗并提示确认停机。用户确认后系统下发停机指令。

```mermaid
flowchart TD
    "进入监控页面" --> "实时参数刷新(5s)"
    "实时参数刷新(5s)" --> "参数是否超限?"
    "参数是否超限?" -->|"温度≥75°C"| "黄色预警通知"
    "参数是否超限?" -->|"温度≥85°C"| "红色报警弹窗"
    "参数是否超限?" -->|"正常"| "实时参数刷新(5s)"
    "红色报警弹窗" --> "确认停机?"
    "确认停机?" -->|"确认"| "下发停机指令"
    "确认停机?" -->|"取消"| "关闭弹窗"
    "下发停机指令" --> "记录告警"
    "黄色预警通知" --> "记录告警"
```

### 3.2 月报生成流程

```mermaid
flowchart TD
    "每月1日自动触发" --> "采集各用风地点数据"
    "采集各用风地点数据" --> "计算实际风量vs需风量"
    "计算实际风量vs需风量" --> "计算有效风量率"
    "计算有效风量率" --> "计算风机效率"
    "计算风机效率" --> "生成月报"
    "生成月报" --> "用户查看/导出PDF"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：工业深蓝(#0A2647) + 安全橙(#F57C00)作为告警强调色
- 辅助色：科技蓝(#144272)、浅灰背景(#F0F2F5)、绿色正常状态(#52C41A)、红色报警(#FF4D4F)
- 按钮风格：Ant Design圆角按钮，主操作使用实心蓝色，危险操作使用红色
- 字体：标题使用思源黑体(Noto Sans SC)，数据使用等宽字体(JetBrains Mono)
- 布局风格：左侧导航栏 + 右侧内容区，顶部状态栏
- 图标风格：线性图标，使用Ant Design Icons

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 通风系统图 | 网络拓扑图 | 深色背景Canvas画布，巷道为带箭头的渐变线段，构筑物为彩色图标节点，悬浮高亮 |
| 通风系统图 | 详情弹窗 | Ant Design Modal，表单式展示详细信息，含参数标签 |
| 主通风机监控 | 参数卡片组 | 5张参数卡片，每张含ECharts迷你仪表盘+数值+单位+状态色 |
| 主通风机监控 | 趋势图 | ECharts折线图，支持时间范围选择，多参数叠加展示 |
| 主通风机监控 | 告警弹窗 | 红色边框Modal，含参数值、超限幅度、确认/取消按钮 |
| 主通风机监控 | 历史告警表格 | Ant Design Table，支持时间选择器+告警级别下拉筛选 |
| 通风月报 | 月报概览 | 统计卡片+对比表格，ECharts柱状对比图 |

| 倒机与反风记录 | 记录列表 | Ant Design Tabs切换倒机/反风，Table展示记录列表 |
| 倒机与反风记录 | 参数对比 | ECharts雷达图/柱状图对比倒机前后参数 |

### 4.3 响应式设计

- 桌面端优先设计，最低支持1366×768分辨率
- 通风系统图支持缩放和拖拽交互
- 监控面板参数卡片自适应排列
